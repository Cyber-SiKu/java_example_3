# 使用面向对象的思想设计一个即时战略游戏的类结构，主要的类和接口如下：

## 1.人口类: 实现可移动接口和可进攻的接口。
+ ### ~~1.1 属性~~:
  + 生命值
  + 攻击力
  + 占用人口个数
  + 消耗资源

+ ### 1.2 子类:
  + 农民 该类可以创建其它人口类如机枪兵、坦克、飞机。

  +  int getDiamond()
